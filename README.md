# ROS2 Differential Drive Robot with 2-Link Arm

## 📌 Overview

This project demonstrates a ROS2 + Gazebo simulation of a 2-wheel differential drive mobile robot with a 2-link robotic arm mounted on top.

The robot is built using URDF/Xacro, visualized in RViz, and simulated in Gazebo. It also includes a camera sensor and demonstrates TF frames and robot modeling concepts.
